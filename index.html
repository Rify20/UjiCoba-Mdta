Tentang Yayasan

Profil Singkat

Yayasan Nurul Hikmah Tugu Liyep berfokus pada pendidikan dan pembinaan karakter, disertai kegiatan sosial untuk mendukung masyarakat sekitar. Halaman ini sengaja dibuat statis, supaya cepat dipakai untuk presentasi ke customer & deploy GitHub Pages.

Brand color: hijau (utama) + oren (aksen) + gradient.
